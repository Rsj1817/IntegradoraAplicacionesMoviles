# SnapRec

## Descripción del Proyecto:
SnapRec es una herramienta de productividad diseñada para capturar ideas al instante. Funciona como una grabadora de voz avanzada que no solo almacena el audio, sino que <strong>transforma automáticamente la voz en texto</strong>, permitiendo a los usuarios crear notas rápidas y precisas sin necesidad de escribir.

Muchas veces necesitamos tomar notas en situaciones donde escribir es lento, incómodo o imposible (como al caminar, conducir o durante una conferencia rápida). Además, las notas de voz tradicionales tienen la desventaja de que no se pueden "leer" ni buscar contenido específico fácilmente.

<strong>SnapRec</strong> soluciona esto eliminando la barrera de la escritura manual, permitiendo gestionar la información de manera más eficiente mediante la transcripción inmediata. Para capturar el audio y procesarlo, la aplicación hace uso del <strong>micrófono</strong> del dispositivo móvil e integra servicios de reconocimiento de voz.
